# 안드로이드 무선 디버깅 설정하는 방법

안드로이드 개발자 옵션에서 디버깅과 무선 디버깅이 활성화되어 있는지 확인합니다.

안드로이드에 고정 IP로 동일한 무선 네트워크에 연결되어 있는지 확인합니다.

연결된 장치들을 확인합니다.

```sh
$ adb devices
```

먼저 adb의 프로토콜과 포트를 지정합니다.

```sh
$ adb tcpip <포트번호>
```

장치를 추가 합니다.

```sh
$ adb connect <ip주소>:<포트번호>
```

다시 연결된 장치들을 확인합니다.

```sh
$ adb devices
```
