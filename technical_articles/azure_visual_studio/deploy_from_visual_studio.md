# Visual Studio를 통한 앱 배포

Visual Studio를 통해 앱을 배포할때 계정과 비밀번호 입력을 요구 합니다. 이때, 계정과 비밀번호는 개발자 계정이 아니라, MicrosoftAppId와 MicrosoftAppPassword 이며 이 정보는 `Azure Portal > Resource (App) > Application Settings` 페이지의 `Application Settings`에 추가를 해야 합니다.
