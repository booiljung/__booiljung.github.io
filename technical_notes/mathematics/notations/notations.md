# Mathematical Notations

## $\arg \min \max$

| Symbols           | Descriptions                 |
| ----------------- | ---------------------------- |
| $\min(f(x))$      | $f(x)$의 최소값              |
| $\max(f(x))$      | $f(x)$의 최대값              |
| $\arg \min(f(x))$ | $f(x)$를 최소로 만드는 $x$값 |
| $\arg \max(f(x))$ | $f(x)$를 최대로 만드는 $x$값 |
