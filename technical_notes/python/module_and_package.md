# 파이썬 모듈과 패키지

파이썬 패키지는 `__init__.py` 파일을 포함 합니다.

아래와 같은 파이썬 모듈들이 있습니다.

```
[]package1
	module11.py
    []package2
		__init__.py
		module21.py
		module22.py
	[]package3
		__init__.py
		module31.py
		module32.py	
```

 `package1`에는 `module11.py`가 있습니다. `package2`와 `package3` 각각 두개의 모듈과 `__init__.py`를 가지고 있습니다.
